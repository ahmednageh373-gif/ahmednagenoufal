# 🎨 دليل مرئي - سير العمل في إدارة المقايسات

## 📊 الصفحات الخمس الرئيسية

```
┌─────────────────────────────────────────────────────────────────┐
│                    🎯 إدارة المقايسات والجدول الزمني            │
└─────────────────────────────────────────────────────────────────┘
           │
           ├── 1️⃣ إدارة المقايسات (BOQManualManager)
           │    └── إدخال يدوي للبنود
           │
           ├── 2️⃣ رفع المقايسات (BOQUploadHub)
           │    └── رفع ملف Excel/CSV + سير عمل تلقائي
           │
           ├── 3️⃣ تحليل المقايسات المرفوعة (BOQUploadAnalyzer) ⭐ جديد
           │    └── تحليل ذكي + تحويل إلى WBS
           │
           ├── 4️⃣ الجدول الزمني (ScheduleManager)
           │    └── عرض وتعديل الجدول الزمني
           │
           └── 5️⃣ تحليل الجدول الزمني (ScheduleAnalysis)
                └── تحليلات متقدمة (Critical Path, Float, Resources)

┌─────────────────────────────────────────────────────────────────┐
│                    💰 إدارة القيمة المكتسبة (EVM)                │
└─────────────────────────────────────────────────────────────────┘
           │
           ├── 6️⃣ لوحة القيمة المكتسبة (EVMIntegratedSystem)
           │    └── مؤشرات PV, EV, AC, CPI, SPI
           │
           ├── 7️⃣ Gantt Chart مع EVM (GanttChartEVM)
           │    └── جدول زمني مرئي مع مؤشرات الأداء
           │
           ├── 8️⃣ التقارير الأسبوعية (WeeklyReportPDF)
           │    └── تقارير PDF للمالك والاستشاري
           │
           └── 9️⃣ نظام التنبيهات التلقائي (AutoAlertSystem)
                └── تنبيهات عند CPI/SPI < 0.90
```
