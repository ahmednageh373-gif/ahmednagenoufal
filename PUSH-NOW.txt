الخطوة 1: احصل على Token
--------------------
افتح هذا الرابط في متصفحك:
https://github.com/settings/tokens/new

املأ:
- Note: AhmedNagehNoufal-Deploy
- Expiration: No expiration
- ✅ حدد: repo (كل الخيارات تحته)
- ✅ حدد: workflow

اضغط "Generate token"
⚠️ انسخ الـ Token فوراً (مثل: ghp_xxxxxxxxxxxx)


الخطوة 2: شغّل هذا الأمر
--------------------
cd /home/user/webapp

git push https://YOUR_TOKEN@github.com/ahmednageh373-gif/ahmednagenoufal.git genspark_ai_developer

⚠️ استبدل YOUR_TOKEN بالـ Token الذي نسخته!


مثال:
------
إذا كان الـ Token: ghp_abc123XYZ789...

الأمر يصبح:
git push https://ghp_abc123XYZ789@github.com/ahmednageh373-gif/ahmednagenoufal.git genspark_ai_developer
